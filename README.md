# Sonic Healthcare (SHL.AX) Analysis Dashboard

A comprehensive financial analysis dashboard for Sonic Healthcare Ltd (ASX: SHL), providing fundamental analysis, green flag scoring, business cycle assessment, and institutional rotation outlook.

## Live Dashboard

**[View Live Dashboard](https://shl-dashboard.vercel.app)** (Update this link after deployment)

## Features

- **Company Overview** - Key metrics, market cap, sector information
- **Morningstar Valuation** - Fair value estimate ($33 AUD), 5-star rating, narrow moat assessment
- **Business Cycle Analysis** - Current mid-cycle recovery phase with 12-18 month outlook
- **Institutional Rotation Assessment** - Factors affecting institutional capital flows
- **Green Flag Scorecard** - 30-point analysis across Income Statement, Balance Sheet, and Cash Flow
- **Interactive Charts** - Revenue trends, margin analysis, return metrics (Chart.js)
- **Historical Data** - 7-year financial summary table (FY2019-FY2025)
- **Analyst Commentary** - Bulls/Bears perspectives from Morningstar

## Key Investment Thesis

### Positives
- 28% discount to Morningstar fair value estimate
- 4.59% dividend yield with 75% payout ratio
- Narrow moat from scale advantages in diagnostics
- Defensive healthcare sector exposure
- Strong cash conversion (OCF/Net Income: 2.54x)

### Considerations
- Post-COVID earnings normalization ongoing
- ROE compressed from 22% (FY21) to 6% (FY25)
- ROIC (4.08%) only modestly above WACC (7%)
- Acquisitive growth strategy execution risk
